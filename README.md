# FUTURE_FS_02 - Client Lead Management System (Mini CRM)

## Overview

This project is a Client Lead Management System (Mini CRM) developed as part of the Future Interns Full Stack Web Development Internship.

The application helps businesses manage customer leads, track lead status, and organize client information efficiently.

## Features

* Add New Leads
* View Lead List
* Update Lead Status
* Edit Lead Information
* Delete Leads
* Search and Filter Leads
* User-Friendly Interface
* Responsive Design

## Technologies Used

* HTML5
* CSS3
* JavaScript
